# Didactic Eureka

Sitio educativo de filosofía y teología, creado por Tomas Lavados Sepúlveda.  

Este repositorio contiene todos los archivos necesarios para el sitio web:

- `_config.yml` – Configuración de Jekyll
- `index.html` – Página principal
- `css/` – Estilos del sitio
- `js/` – Scripts
- `img/` – Imágenes y recursos gráficos

## Objetivo

Proveer material didáctico de filosofía y teología de manera estructurada y accesible.
